# 05. Shaft (축)

**역할:** Turbine과 Compressor를 기계적으로 연결. 관성 미분방정식(ODE)으로 엔진 과도 현상 모델링.

---

## 모델 개요

| 항목 | 내용 |
|------|------|
| 모델링 언어 | VHDL-AMS |
| 입력 | mech_rv (Turbine 회전동력) |
| 출력 | 회전동력 → Compressor, 프로펠러 부하 |
| 특징 | 5개 컴포넌트 중 **유일한 ODE 기반** — 과도 응답 모델링 담당 |

## 핵심 파라미터

| 파라미터 | 값 | 설명 |
|----------|----|------|
| inertia | 1.0 kg·m² | 축 관성모멘트 |
| omega0 | 0.0 rad/s | 초기 각속도 |
| across/through | omega / tau | 각속도 / 토크 변수 쌍 |

## 동역학 방정식

```
J * dω/dt = W_turb - W_comp - W_prop
```

- J: 관성모멘트, ω: 각속도
- W_turb: 터빈 입력 동력, W_comp: 압축기 소비 동력, W_prop: 프로펠러 부하
