# 05. Shaft (축)

**역할:** Turbine과 Compressor를 기계적으로 연결. 관성 미분방정식(ODE)으로 엔진 과도 현상 모델링.

---

## 모델 개요

| 항목 | 내용 |
|------|------|
| 모델링 언어 | VHDL-AMS |
| 입력 | mech_rv (Turbine 회전동력) |
| 출력 | 회전동력 → Compressor, 프로펠러 부하 |
| 특징 | 5개 컴포넌트 중 **유일한 ODE 기반** — 과도 응답 모델링 담당 |

## 핵심 파라미터

| 파라미터 | 값 | 설명 |
|----------|----|------|
| inertia | 1.0 kg·m² | 축 관성모멘트 |
| omega0 | 0.0 rad/s | 초기 각속도 |
| across/through | omega / tau | 각속도 / 토크 변수 쌍 |

> **참고 (초기조건 vs 정상상태):** omega0=0.0 rad/s는 시뮬레이션 초기조건(t=0)이며, 동역학 방정식(J·dω/dt = W_turb - W_comp - W_prop)을 적분하면 정상상태 각속도는 약 4,370 rad/s(≈41,730 RPM)로 수렴합니다. 이는 Honeywell TPE331-10 공식 스펙의 gas generator(터빈-압축기) 축 회전수(41,730 RPM)와 일치하는 값입니다. 감속기어를 거쳐 구동되는 프로펠러 출력축 회전수(2,000 RPM, 감속비 ≈20.9:1)와는 서로 다른 축의 값이므로 혼동하지 않도록 주의가 필요합니다.

## 동역학 방정식

```
J * dω/dt = W_turb - W_comp - W_prop
```

- J: 관성모멘트, ω: 각속도
- W_turb: 터빈 입력 동력, W_comp: 압축기 소비 동력, W_prop: 프로펠러 부하
